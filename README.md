# Fraud Thesis Project

Master thesis project on fraud detection using imbalanced learning techniques.

## Main goal
Implement a simplified version of the FROID-inspired framework for highly imbalanced fraud detection.

## Planned methods
- PCA
- LOF
- Isolation Forest
- Random Forest
- LightGBM
- SMOTE for baseline comparison

## Project structure
- `data/raw/`: raw datasets
- `data/processed/`: processed datasets
- `notebooks/`: exploratory analysis
- `src/`: source code
- `results/figures/`: plots
- `results/tables/`: result tables
- `thesis/`: thesis writing files
